# Custom Agent Rules & Command Phrases

This file contains persistent, project-specific rules, mappings, and command phrases for AI Coding Agents working on this workspace.

## Color Mappings & Command Phrases

- **Command Phrase**: `"app green"` / `"make app green"`
  - **Color**: `#62f507` (Rosie green)
  - **Description**: Whenever the user requests to make elements green, color the app green, or style elements using the green theme, use the vibrant Rosie green color hex `#62f507` instead of standard or forest greens.
